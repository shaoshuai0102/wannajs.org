Introducing Wanna

Use Wanna if you wanna express yourself with a blog.

Make it natural.

It's super easy and natural to use Wanna. All you want to do with a blog is writing, viewing and publishing, how about trying "wanna write, wanna view, wanna publish"? You will really like it.

Make it beautiful.

There are designers and developers making various themes for you. Especially if your are a web developer, making a theme is super easy with a template engine like mustache, handlbars and jade.

Writing is easy.

Wordpress is great, but writing in a web editor or any other rich text editor made us really suffering. Now here comes markdown! It's the new way to write. Focus on the content, let Wanna do the rest for you.
